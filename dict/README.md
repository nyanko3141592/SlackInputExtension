# 辞書サンプル

popup の「辞書」タブから取り込んで使えるサンプル CSV です。

## 含まれるファイル

| ファイル                    | 内容                                                                                                       |
| --------------------------- | ---------------------------------------------------------------------------------------------------------- |
| `sample-business-terms.csv` | エンタープライズ営業で頻出する一般的なセールス用語・ビジネス略語・ツール名 (FS / KA / MRR / Salesforce 等) |

## 取り込み方

1. Chrome の拡張機能アイコンをクリック → **辞書** タブ
2. 📥 **取り込み** ボタンから CSV を選択
3. 既存の辞書とマージ (同じ `reading` があれば上書き)

## フォーマット

```csv
reading,display,note
ふぃーるどせーるす,FS,Field Sales 略称
```

- `reading`: AI が誤変換しがちな表記 (仮名・発音そのまま等)
- `display`: 出力させたい正式な表記
- `note`: メモ (任意)

UTF-8 エンコーディング、ヘッダー必須。
旧版エクスポート (`from,to,note`) も読み込めます。

## 注意

社内固有の名称 (製品名、サービス名、社員名など) は別途、内部ドキュメントに直接書いた CSV を取り込むことを推奨します。
このリポジトリには企業・個人を特定できる情報を含めません。
