# ✨ Slack AI Rewriter

Slack の Web 版メッセージ入力欄を **Google Gemini AI** でリライト・音声入力できる Chrome 拡張機能。

- 🙇 丁寧にリライト / 🧹 整理 / 🗂️ 構造化 / ✂️ 簡潔化 / 😊 カジュアル化 / • 箇条書き / 🌐 英訳
- 🎤 音声入力 (Slack 入力欄に本文として追記)
- 📖 辞書機能 (読み → 表示の自動置換、CSV import/export)
- 📌 Slack mrkdwn 完全準拠 (`*太字*` `_斜体_` `~取消~` `• 箇条書き`)
- ⌨️ キーボードショートカット (`Alt+Shift+A` / `R` / `1`〜`9`)
