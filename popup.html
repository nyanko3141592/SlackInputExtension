    使い方

    1. API 設定タブで Gemini API キーを保存。
    2. Slack を開いて、メッセージを入力欄に書く。
    3. 入力欄の右上に ✨ AI ボタンが表示されるのでクリック。
    4. プロンプトを選ぶとリライト結果がプレビューされる。
    5. 編集 → 適用 で入力欄に置換。

    Slack Markdown のおすすめ設定

    Slack の「環境設定 → 詳細設定 → メッセージ作成」で 「マークアップでメッセージをフォーマットする」 を ON にすると、*text* 等の出力がそのまま太字として送信されます（推奨）。

    対応する Slack mrkdwn

    • *太字* → 太字
    • _斜体_ → 斜体
    • ~取り消し線~
    • `コード`
    • > 引用
    • • 箇条書き
